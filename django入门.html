4. 模型 __str__ 方法

__str__ 返回值是模型在 admin 里面的显示值

class Artical(models.Model):
    artical_id = models.AutoField(primary_key=True)
    title = models.TextField()
    bref_content = models.TextField()
    content = models.TextField()
    publish_date = models.DateTimeField(auto_now=True)

    def __str__(self):
        return self.title

2. 博客首页

blog/views.py

from django.shortcuts import render

def get_articals(request):
    articals = Artical.objects.all()
    return render(request, 'blog/index.html', {"articals": articals})

blog/urls.py

urlpatterns = [
    path('get_articals', blog.views.get_articals)
]

blog/templates/index.html

<!DOCTYPE html>
